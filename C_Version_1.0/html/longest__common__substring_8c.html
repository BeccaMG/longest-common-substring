void display_result ( char *  result)

Receives an array of char with the LCS result and show it in stdout.

display_result

Parameters
resultResult to be processed and displayed.

The result should be a array of characters holding the longest substring between the two input files.

If the array is not empty then the output will follow this format:

The longest substring between [Input File 1] and [Input File 2] is "[longest substring]".

Otherwise, the output would be No common substrings

If the return value is asigned to a variable outside the function, there's no need of allocating memory for it since the function does it.
